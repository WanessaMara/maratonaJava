# Estudos de Java - Fundamentos
Repositório criado com o objetivo de consolidar meus estudos em Java, passando por lógica de programação, estrutura de dados básicas e Programação Orientada a Objetos (POO)

## 🚀 Objetivo
Desenvolver uma base sólida em Java para atuação como desenvolvedora backend, focando em:
* Lógica de programação
* Resolução de problemas
* Estrutura de dados
* Boas práticas de código

## ⚙️ Conteúdos Abordados
* Lógica (Condicionais, Estrutura de repetição, Operadores lógicos
* Arrays
* Métodos e Modularização
* Programação Orientada a Objeto (POO)
* Estrutura de dados (pilha, fila)
